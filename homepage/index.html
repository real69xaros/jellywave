Frequently Asked Questions

Does it work externally behind Cloudflare?

Yes. Simply reverse proxy to port `1075` using Cloudflare Tunnels or Nginx proxy manager.

Can multiple users log in?

Yes! Any registered user can browse the platform's library securely based on their account.
